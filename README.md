### Projeto clone-tabnews

O projeto em questão é focado na recriação do site https://www.tabnews.com.br com o objetivo de aprender e exercitar os fundamentos da **programação back-end** para criação de _API_, bem como suas funcionalidades e escopos. O projeto também visa aplicar os conceitos estudados de forma prática e como sua implementação impacta dentro do próprio projeto.

## Créditos

Todo o projeto está sendo desenvolvido a partir das aulas **excelentes** e muito bem **estruturadas** do https://curso.dev.
